java university project

## Usage

### Compilation

On linux unix system use compile script, to create 2 jar (cient, server)

```viml
./compile.sh
```

### Execution

Run server before client application

server
```viml
java -jar InterfaceServer.jar
```
#[server](server.png)

client
```viml
java -jar InterfaceClient.jar
```
